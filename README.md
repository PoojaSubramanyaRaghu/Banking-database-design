# Banking Database Design — MySQL

A fully normalized relational banking database built in MySQL covering core banking operations including customer management, accounts, transactions, loans, and cards.

## Features
- 10 relational tables with foreign key constraints
- Stored procedures for customer management, loan details, and account statements
- Triggers for automatic balance updates on transactions
- Views for customer account summaries, loan payment tracking, and branch employee lists
- Index optimization for query performance
- Role-based access control across 6 user roles: Admin, Branch Manager, Teller, Loan Officer, Customer Service, and Auditor
- Audit logging for all system actions

## Tech Stack
MySQL, SQL, Relational Database Design, RBAC
